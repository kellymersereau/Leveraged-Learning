Grid Alignment

Aligning the grid in a case where the number of tracks do not equal the size of the container.

Initial values:

  • align-content: start
  • justify-content: start

The white area below is our grid container. It is 800x800 pixels. Our four column and four row track grid is in total 630x630 pixels including the gaps.

By default it appears top left of the container.
